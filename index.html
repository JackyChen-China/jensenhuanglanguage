Setup Guide
EmailJS 配置指南
免费版每月 200 封 · 5分钟完成配置
📋 配置步骤（共4步）
1
注册 EmailJS 免费账号
打开 emailjs.com，点击 Sign Up 注册免费账号（每月200封免费）
2
添加 QQ 邮箱 Email Service
登录后 → Email Services → Add New Service → 选择 QQ Mail
填入你的 QQ 邮箱和授权码（授权码获取：QQ邮箱设置 → 账户 → 开启SMTP服务）
点击 Connect Account，记下生成的 Service ID（格式：service_xxxxxxx）
3
创建邮件模板 Email Template
Email Templates → Create New Template
Subject 填：{{subject}}
Content 填：{{message}}
To Email 填：{{to_email}}
保存后记下 Template ID（格式：template_xxxxxxx）
4
获取 Public Key 并填入本页面
Account → API Keys → 复制 Public Key
回到「邮件订阅设置」页面，填入 Service ID、Template ID、Public Key，点击「测试发送」验证
⏰ 关于定时发送
本工具运行在浏览器中，网页需要保持打开才能触发定时任务。
建议在电脑上保持浏览器标签页开着，每天早上 07:00 会自动生成并发送邮件。

如需真正的后台定时发送（无需开着网页），可将此工具部署到有后端的平台如 Railway 或 Render，告诉我你需要这个功能即可。
